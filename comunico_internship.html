NOMESTUDIO - Graphic studio website
INFORMATICS EXAM PROJECT, Polimi.

NOME STUDIO

Double Css

Politecnico di milano.
IT elements and Computer network.

2018


Motion Graphics

The directives for the final exam of the course presented an obvious paradox: creating a personal website but without entering any personal data.
Starting from that I invented this fictitious studio called “NOMESTUDIO”, a term used in the design phase as a simple place marker.
So I created a site that could be used both as a personal portfolio or as a web platform for a small graphic studio.
Another requirement was to equip the site with two different CSSs that differed completely in the atmosphere transmitted.
For this reason, by clicking in the hamburger menu at the top right, you have the possibility to choose between 'White Style' (by default) and 'Dark Side'.
